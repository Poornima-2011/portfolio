# Personal Portfolio Website

A modern and responsive personal portfolio website developed to showcase my profile, technical skills, projects, and contact details. This project is designed with an attractive user interface and integrated with a Spring Boot backend for handling contact form submissions.

## 🚀 Live Demo

🌐 Portfolio Website: https://Poornima-2011.github.io/portfolio/

## 🚀 Features

- Modern responsive design for desktop and mobile devices
- Animated gradient background and smooth scrolling
- About Me section with professional summary
- Skills section displaying technical expertise
- Projects section with GitHub source code links
- Contact form integrated with Spring Boot REST API
- Resume download button
- GitHub and LinkedIn profile links
- Clean and recruiter-friendly UI

## 🛠️ Technologies Used

### Frontend
- HTML5
- CSS3
- JavaScript

### Backend
- Java
- Spring Boot
- Spring Web
- Spring Data JPA
- MySQL

📬 Contact Form Workflow
1. User enters name, email, and message.
2. JavaScript sends data using Fetch API.
3. Spring Boot REST API receives request.
4. Data is stored in MySQL database.
5. Success message shown to user.
